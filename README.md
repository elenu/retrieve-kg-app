# regflux-omnipath-app

![Browser](https://github.com/elenu/retrieve-omnipath-app/blob/main/App_Sample.png?raw=true)

## Overview

The `retrieve-omnipath-app` is a Shiny application designed to analyze gene interactions using data from the Omnipath interactions table. Users can select source and target gene symbols from the provided dataset to explore their interactions (published by Omnipath).

## Installation
To install the necessary packages, run the following command in R:

```R
install.packages(c("shiny", "dplyr", "ggplot2"))
```

## Usage

You can use this Shiny app to explore Omnipath gene interactions. Select source/target gene symbols and visualize reported interactions.

1. Install CRAN packages (bash)

```
Rscript -e 'install.packages(c("shiny","readr","dplyr","stringr","visNetwork","igraph","bslib","testthat"), repos="https://cloud.r-project.org")'
```

2. Install Bioconductor OmnipathR (required by R/download_omnipath.R)

```
Rscript -e 'if (!requireNamespace("BiocManager", quietly=TRUE)) install.packages("BiocManager", repos="https://cloud.r-project.org"); BiocManager::install("OmnipathR")'
```

3. Download Omnipath interactions (run from repo root)

```
Rscript --vanilla R/download_omnipath.R
```

4. Launch the application by running the `app.R` file in R.

```r
source("app.R")
```

or in bash:

```
R -e 'shiny::runApp(".", launch.browser = TRUE)'
```

5. Use the gene selection interface to choose the source and target gene symbols from the dropdown menus.

6. Explore the interactions and visualizations generated by the app. Run tests with bash (or in r):

```
Rscript -e 'if (!requireNamespace("testthat", quietly=TRUE)) install.packages("testthat", repos="https://cloud.r-project.org"); testthat::test_dir("tests")'
```

## Data

he application uses the `omnipath_interactions_table.csv` file located in the `data` directory. This file contains gene interaction data that the app processes to provide insights based on user selections. The current file has been downloaded from Omnipath and cut in order to ease the loading to github.

### Downloading Omnipath interactions

A helper script is provided to download the latest Omnipath interactions using the OmnipathR Bioconductor package:

- Script: `R/download_omnipath.R`
- Usage:
  - From shell: `Rscript R/download_omnipath.R`
  - From R: `source("R/download_omnipath.R")`

The script will create `data/omnipath_interactions_table.csv` (creating the `data/` directory if missing).

## Notes
- app.R expects column names `source_genesymbol` and `target_genesymbol` in the CSV.
- Recommended example: GRB2 gene as source symbol, and SOS1 as target symbol.
